## Notes
- The frontend uses the Node ecosystem (`npm` / `yarn`). The `package.json` in
  the repo contains minimal dependency hints; use the list above to add any
  libraries you need for features like camera, ML, or barcode scanning.
- The backend package list is a recommended starting point for a FastAPI
  server (optional). Trim packages to what you actually use.
